####Steps to reproduce:
- Have admin permissions in a server 
- Open Server Settings -> Overview 
- Have another person/account that also has admin permissions make changes on the overview page (eg. changing server name, setting an afk channel, etc). 
- Click ''Save Changes'' on the alternative account 
- Observe how the ''Unsaved changes''-warning pops up on the original account, that hasn't made any changes what so ever.

####Expected result:
No warning should appear on the original account.
####Actual result:
The ''Unsaved Changes''-warning appears.

####System settings:
