[Archived] Middle clicking on a text channel in the channel list opens the web version of whichever type of client you're using in the browser on that channel, which can cause confusion on misclicks (https://trello.com/c/jHVl1z5j)

P3: Will get fixed eventually

Created: 2016-10-23T11:01:17.439Z
Latest Update: 2017-04-21T22:42:18.646Z

####Steps to reproduce:
- Find any text channel you can access
- Middle click on the channel in the channel list on the left

####Expected Result: 
Nothing, or changing into autoscroll mode as it does in the user list
####Actual Result: 
Opens up your default browser to the discord web version on the channel you middle-clicked
####Platform 
Win10 Pro x64, tested on Canary, PTB and Release client versions
