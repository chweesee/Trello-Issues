####Steps to reproduce: 
- Be in 2 servers where you cannot use the custom emotes anywhere (to put it simply, 2 servers that have custom emotes/emojis) ( Example: Server 1; `:NotLikeThis:` :: Server 2; `:whentm:` ) 
- In one of the servers (for the example, server 2), use/type an emoji/emote from both servers (make sure to type it out correctly) 
- Send the message 

####Expected result:
 The custom emoji from the other server will be in text while the custom emoji from the server you sent it will be displayed as an emoji (`:NotLikeThis:` is in text while `:whentm:` is displayed as an emoji) 
####Actual result:
 Both custom emojis are displayed in text (`:NotLikeThis: :whentm:`) 
####System settings:
 Windows 10, Discord Canary
