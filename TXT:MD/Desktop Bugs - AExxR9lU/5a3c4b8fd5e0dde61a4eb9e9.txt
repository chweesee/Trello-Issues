Reported by Lt_Lemming#5217

####Steps to reproduce:
- Open discord with audio device attached 
- with audio device is set to anything other than 16bit/44.1KHz you can see audio on the input device, but Discord will not receive audio 
- open windows "Sound" control panel. 
- go to "Recording" tab 
- right click on audio device and select "Properties" 
- click on "Advanced" tab. 
- Change "Default Format" drop down to "16bit/44.1KHz (CD Quality)" 
- Discord will now receive audio from the input device.

####Expected result:
Discord should be able to receive audio from the device regardless of format.
####Actual result:
Discord receives no audio from device until format is set to "CD Quality"
####Client settings:
Release Channel: stable, Build Number: 8399, Version Hash: 68c4e879467d225279f227c6a5f1c8866b4d96e2
####System settings:
Windows 10 Pro Build 1703 & Windows 10 Enterprise LTSB Build 1602. Focusrite Scarlett 2i2 and 2i4 with latest drivers.
