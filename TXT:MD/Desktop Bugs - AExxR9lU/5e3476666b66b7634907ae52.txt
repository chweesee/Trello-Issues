When the "Space between message groups" option is set to 0px, it gets reset to the default value upon restarting Discord (https://trello.com/c/p5PK0ib0)

Reported by GrimReapeR#3533

####Steps to reproduce:Go into Settings. 
- Go to Appearance. 
- Make sure your "Message display" option is set to Cozy, because Compact is not affected by this bug. 
- Set the "Space between message groups" option to the lowest value (0px). IT MUST BE 0px, other values don't seem to be affected by this bug. 
- Restart Discord. 
- Observe.

####Expected result:
The "Space between message groups" option should stay on 0px.
####Actual result:
The "Space between message groups" option gets reset to the default value.
