Discord uses bcrypt hashing for passwords. The 128 character limit for passwords is too high for this algorithm. (https://trello.com/c/RBp61Pnk)

None

Created: 2019-07-01T04:20:36.918Z
Latest Update: 2020-07-16T21:26:04.364Z

Reported by LogalDeveloper#6919

####Steps to reproduce:
- Set your password to make use of the full 128 character limit. 
- Log out and log back in, except when using the password, trim a few characters off the end and attempt to log in. 
- Notice how it still works. Next, change to a 72 character password and do the same thing. 
- Notice how this time, it prevents the log in.

####Expected result:
All 128 characters should be factored in when hashing the password.
####Actual result:
Only 72 characters are factored in. This is a limit in bcrypt, which only supports up to 72 characters.

####System settings:
Ubuntu 18.04.2 LTS - Chromium 75.0.3770.90
