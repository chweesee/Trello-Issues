####Steps to reproduce:
- Have a set of GIF emotes that stack on each other (two lines). I.E. (1,2 (shift,enter)3,4) 
- Post it as message #1 all in the correct order. 
- Send message #2 with the same emotes in one line (1,2,3,4,) 
- Notice it effects rotation of message #1 
- Edit message #2 for it to be identical to message #1. 
- Notice rotation change again. 
- Edit message #2 again and click cancel. 
- message #1 and 2 sync

####Expected result:
Editing a message shouldn't effect a separate message's GIF emote rotation.
####Actual result:
If effects separate messages, causing things to get out of sync.

Canary 38181
####System settings:
Win 10 Pro x64 (1809)
