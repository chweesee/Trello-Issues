####Steps to reproduce:
- Make a private channel locked with a role 
- Asign the user the role to access the private channel 
- Let the user go to the private channel and open search 
- Search up a keyword 
- Let the user leave the server 
- Let the user rejoin the server

####Expected result:
The search window shouldn't still be open when you rejoin the server and allow you to see messages in the private channel through the search window
####Actual result:
You have the search window still open and you can see the messages in the private channel. (not jump though)

Discord Canary 0.0.169 (Build 6746)
####System settings:
Windows 10 Pro 64-bit (Build 1607)
