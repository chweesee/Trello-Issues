####Steps to reproduce:
- Run a streamable game 
- Join voice channel in any server 
- Switch to server with Streaming feature 
- Click the Stream button above user info 
- Click 'Go Live' 
- Switch back to the voice server

####Expected result:
Upon clicking 'Go Live' it should begin streaming, or alert you that you're in a different server.
####Actual result:
Everyone in the call will hear the 'Live' sound ping and the 'Live' tag will appear quickly before disappearing.
