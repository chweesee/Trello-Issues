####Steps to reproduce:
- Make sure your font size is set to 16px in appearance -> chat font scaling 
- Open the custom status modal and select the "Clear After" drop down 
- Try to scroll 
- Repeat these steps with 15px and 18px chat font scaling

Changing the chat font scaling has no impact on the "Clear After" dropdown in Custom Status

Changing the chat font scaling changes the length of the dropdown, cutting of words at lower chat font scaling and making scrolling necessary, and there is an unnecessary scrollbar at the default chat font scaling
