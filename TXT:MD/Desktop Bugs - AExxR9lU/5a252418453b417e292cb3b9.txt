####Steps to reproduce:
- Be on a server where you have permission to manage roles. 
- Find or create a role and don't change it from its default color. 
- Click on the user so that the profile card shows up. 
- Hover over the default color role and a colored role.

####Expected result:
Roles with default color and roles with a color should both highlight when moused over.
####Actual result:
Only the default colored role highlights when moused over.
####Client settings:
Canary 0.0.181
####System settings:
Windows 10 Home x64 Version 1703
