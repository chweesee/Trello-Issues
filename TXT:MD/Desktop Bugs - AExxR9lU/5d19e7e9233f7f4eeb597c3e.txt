Category creation modal does not close when getting a desktop notification and clicking it. (https://trello.com/c/6LEbtwCF)

None

Created: 2019-07-01T11:00:57.857Z
Latest Update: 2020-07-16T21:26:04.371Z

Reported by Skyzar#0001

####Steps to reproduce:
- Have desktop notifications turned on. 
- Go to a server where you can create a category. 
- Create a category (just open the modal and don't actually create one). 
- Get a DM from someone so that you get a desktop notification. 
- Click on the notification. 
- Your client will change to the DM page but the modal for creating the category will still be open.

####Expected result:
The "create category" modal is supposed to close.
####Actual result:
It doesn't close and you have to press cancel to close it.
####Client settings:
Release Channel: stable, Build Number: 40273, Version Hash: 7ee65925bd613109d7311da93f9d8f557762ff77
####System settings:
Windows 10 Pro, Version 1809, Build 17763.557
