Reported by Pascal#0929

####Steps to reproduce:
- Open any muted (or mute any) server with new unread messages 
- Right click the server icon and untick "Mute Server" 
- Change the server and observe the server icon for the server with unread messages

####Expected result:
Unread marker for the server and a badge for the Discord icon for the unread messages should instantly be shown.
####Actual result:
You have to reopen the server, switch channels in the server or wait for other new message to see it.
####Client settings:
Discord Canary 0.0.191 (Build 15598)
####System settings:
MacBook Pro 13" (2016), macOS High Sierra 10.13.4
