Reported by Stig124#1591

####Steps to reproduce:
- Be sure that Aero Snap is enabled (Settings-System-Multitasking-[Enable everything under Working with multiple windows]

- Launch Discord and dock it (using Aero Snap) to one side (the bug happens in both the 1-a-side config and 2-a-side config)

- Write something that takes more than one line when docked

- Put Discord on normal position with Snap

####Expected result:
The message bar should resize correctly when the window is resized
####Actual result:
The message bar extend in height, but as soon as you do a new line or leave the channel the message bar resizes correctl

Release Channel: canary, Build Number: 50509
####System settings:
Windows 10, 1909, 18363.476
