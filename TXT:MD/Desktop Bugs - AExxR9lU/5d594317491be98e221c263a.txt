Published messages from announcement channels do not change to "[Original Message Deleted]" when the original message is deleted via the bulk deletion endpoint (https://trello.com/c/sJLbyFTN)

Latest Update: 2020-07-16T21:26:04.880Z

Reported by webhp#1337

####Steps to reproduce:
- Have a server with an announcement channel 
- Follow that channel in another channel 
- Send a message in the announcement channel and publish it 
- Using a bot or manually with a bot token, delete the message with the bulk deletion endpoint

####Expected result:
The message should be edited to say "[Original Message Deleted]" in all channels were it was crossposted in
####Actual result:
It does not edit

Discord Canary v0.0.254 Build 43884
####System settings:
Windows 10 Pro x64 v1903 Build 18912.1001
