None

Latest Update: 2020-07-16T21:26:04.441Z

Reported by defracted#8618

####Steps to reproduce:
- Click on «Create Server» button 
- Type in the name input field something with at least two spaces using Latin symbols (e.g. `asd asd asd`) 
- Now type in the field something with at least two spaces using Cyrillic symbols (e.g. `фыв фыв фыв`) 
- Compare the server icons

####Expected result:
Server name initials in the icon should show only first letter of each word with Cyrillic symbols
####Actual result:
Server name initials in the icon shows whole word instead of the first letter
####Client settings:
Stable 41102
####System settings:
Windows 10 Pro (1903)
