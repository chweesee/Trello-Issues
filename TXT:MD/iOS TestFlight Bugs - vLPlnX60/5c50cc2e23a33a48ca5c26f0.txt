####Steps to reproduce:
- Enable 2FA and copy one of your backup codes 
- Logout from the app 
- Login again 
- When you get the 2FA screen, tap on the ellipsis on the top right and tap on `Backup Codes` 
- Paste or type in one of the backup code 
- Tap on the OK button 
- Observe

####Expected result:
If the backup is valid and unused, it should log you in, or show an error saying the code is invalid
####Actual result:
Upon clicking on the OK button, the app crashes, rendering it impossible for you to use your backup code to login.
####Client settings:
TestFlight Build 2.4.4 (10973)
####System settings:
iPhone 7 iOS 12.1
