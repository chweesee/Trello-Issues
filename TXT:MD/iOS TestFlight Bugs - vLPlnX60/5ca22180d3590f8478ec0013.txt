####Steps to reproduce:
- Send a message in any channel 
- Now get only view messages permission but not send messages permission for this channel  (read-only channel) 
- Long hold the message you have sent there and try to edit it

####Expected result:
Edit message option shouldn't appear in read-only channels or you should get a permission error (similar to Desktop).
####Actual result:
You get no permission error message and can edit the message by marking it as a spoiler for example (typing in the text field doesn't work).
####Client settings:
TestFlight 2.5.2 (13014)
####System settings:
iPhone 8 Plus & iPad Pro 11" - iOS 12.2
