####Steps to reproduce:
- Go to any server that contains a lot of channels and categories (The Discord Testers server can be used). 
- Hold down the server icon, in the same fashion as when you want to rearrange the server icons. 
- Scroll up and down the channel list, without letting go of the server icon.  
- Observe.

####Expected result:
The channel list should appear fine OR you shouldn't even be able to scroll up and down the channel list when moving the server icon at allz
####Actual result:
You are able to move the channel list while going up or down the channel list, and the channel list faces odd and random movements.
####Client settings:
TestFlight 3.0.2 (15253)
####System settings:
iPhone 8, iOS 12.3
