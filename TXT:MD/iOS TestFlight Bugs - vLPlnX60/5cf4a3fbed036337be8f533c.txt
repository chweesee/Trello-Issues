Reported by bryant#1011

####Steps to reproduce:
- Be on iOS 12.3.1 or 12.4 Public beta 
- Have the keyboard open 
- Select any message in the server with the keyboard open. 
- React to the message 
- observe

####Expected result:
After you react to the message, the keyboard remains up but at the last message. To allow you to continue sending messages.
####Actual result:
After you react to a message, the keyboard stays up and cuts off the message bar including 2-3 messages. You are able to type but can not send the message. You have to close the keyboard and re-open to be able to send the message.
####Client settings:
TestFlight 3.0.2(15283)
####System settings:
iPhone X, iOS 12.3.1/iPad 9.7”, iOS 12.4(Current Public Beta)
