None

Latest Update: 2017-10-17T21:13:52.546Z

Steps to Reproduce:

1)  In device Settings, select the Discord application.
2)  Turn Photos permission OFF.
3)  Turn Camera permission ON.
4)  Progress to any instance of chat within Discord.
5)  Tap on the paper clip icon to start the upload flow.
6)  Select Take Photo.
7)  Take a picture.
8)  The "Use Photo" button appears.

Buggy event:  The application can no longer be interacted with.  At this point, the user must close and relaunch the application.
Expected:  User should be able to upload image.
