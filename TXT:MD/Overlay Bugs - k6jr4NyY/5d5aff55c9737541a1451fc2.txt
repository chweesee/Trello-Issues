When you go to another server through the miniprofile of the news following webhook, you cannot click back on the server icon you were originally in. (https://trello.com/c/gJDnBAZ3)

Reported by Isabel#0002

####Steps to reproduce:
- Go to a server that has an announcements channel in the normal client. 
- Follow the announcements channel to a channel in another server. (also in client) 
- In the overlay, go into the channel you followed the announcements channel to. 
- In the overlay, click the username of the following webhook and click "View Server". 
- Then try to click the server icon in the server list to go back to the server you were just in.

####Expected result:
You switch back to the server.
####Actual result:
You are stuck in any other server except the previous server. To fix it you can click on any other server then the original server's icon.
####Client settings:
: Canary 43738
####System settings:
Windows 10 Home x64 (1903)
