####Steps to reproduce:
- Connect a second screen to the HDMI port 
- Use Winkey + P to open the project menu 
- Select "Second screen only" 
- Open Discord desktop client 
- Launch any game that has overlay enabled 
- Close game and Discord completely 
- Disconnect second screen 
- Reopen Discord client 
- Launch the same game

####Expected result:
Discord Overlay should appear on the primary screen in-game.
####Actual result:
Discord Overlay does not appear. It is still locked on the second display, even though it is not connected.
####Client settings:
Discord Canary, Build Number: 16234
####System settings:
Windows 10 x64, Intel HD Graphics 530, NVIDIA GeForce GTX 960M
