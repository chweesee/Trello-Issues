[Archived] Copying text from a website that is wrapped in a centered h1 tag and pasting it in Discord causes the cursor and all text you type to be centered (https://trello.com/c/oH9iWB4I)

None

Created: 2019-12-07T00:05:20.105Z
Latest Update: 2020-02-20T21:32:07.889Z

Reported by Alex1304#9704

####Steps to reproduce:
- Open a web page that has text wrapped in a `<h1 style="text-align: center;">` HTML tag 
- Copy the text that is inside the tag 
- Paste it in Discord in the message box

####Expected result:
It should paste text properly
####Actual result:
The centered property is carried over the textbox, causing all text you paste/type to be centered. Removing pasted text dorsn't fix it, you need to switch channel and switch back to remove the centered effect. A workaround to avoid this is to paste text as raw
####Client settings:
Discord version 9.9.4
####System settings:
Huawei Mate 9 Android 9.0
