[Archived] In Server Settings > Overview, if you originally had welcome messages disabled, enabling them won't unlock the two switches for server boosts and welcome messages unless you save first (and vice versa) (https://trello.com/c/An08A1Oe)

None

Created: 2019-06-07T12:07:17.839Z
Latest Update: 2019-08-16T21:24:16.351Z

Reported by Alex1304#9704

####Steps to reproduce:
- Go to Server Settings > Overview  
- Disable welcome messages if not already 
- Save 
- Now enable them again, but don't save yet  
- Notice the switches are still greyed out 
- Press Save. Now the switches will unlock

####Expected result:
The two switches "Send random message when someone joins" and "Send message when someone boosts the server" should immediately lock/unlock upon selecting a new channel in the dropdown menu, without requiring you to save
####Actual result:
Changing the welcome messages channel in the dropdown menu won't lock/unlock the switches automatically, you need to save first and then re-save after you changed the switches.
####Client settings:
Discord Alpha 9.0.9
####System settings:
Huawei Mate 9 Android 9.0
