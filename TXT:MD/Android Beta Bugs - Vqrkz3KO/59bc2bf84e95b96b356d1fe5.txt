####Steps to reproduce:
- Create a new server (or go to one where you're owner/admin) 
- Go to server settings, then to roles section 
- Create a new role 
- Open the color picker; Observe the current color of role 
- Change the role color to something else 
- Open the color picker again; Observe: the last color from picker (the default one which the role had when it was created) doesn't show up anymore

The color should be available in the color picker
####Actual result:
The color doesn't show up anymore in the color picker
