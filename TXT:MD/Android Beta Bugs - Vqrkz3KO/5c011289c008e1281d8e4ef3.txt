####Steps to reproduce:
- be on a server where you have move user and server deafen permissions 
- long press on a user in a voice chat 
- observe the user menu that appears

####Expected result:
the 'move user' and 'server deafen' buttons should appear in red.
####Actual result:
no red action buttons appear. if the server mute permission is granted, all 3 of the buttons appear.  conversely, if a user has server mute permissions, but not move or deafen perms, all 3 buttons will appear anyway.
####Client settings:
Android alpha version 802.
####System settings:
Samsung Galaxy a5, Android 8
