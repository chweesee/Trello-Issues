Reported by Granny#9999

####Steps to reproduce:
- Set up 2FA on your Discord account 
- Enable SMS Backup on your mobile device 
- When the "Enter the SMS verification code we sent..." screen appears, press the X button at the top-right of the screen 
- Close and relaunch discord 
- Open the My Account screen 
- Toggle on SMS Backup 
- Enter your password

####Expected result:
Discord should send an SMS verification message, and ask you for the code before enabling SMS Backup.
####Actual result:
No code is sent. SMS Backup is enabled without ever sending a code to your mobile phone.
####Client settings:
Android Alpha 9.0.9
####System settings:
Huawei P20 Lite, EMUI 9.0
