[Archived] When you allow Discord to use the Nearby service while you're using the nearby scanning makes it detect no nearby device. (https://trello.com/c/NYnAl1pC)

P3 - Will be fixed eventually

Created: 2019-03-12T18:39:54.028Z
Latest Update: 2020-02-20T21:34:06.548Z

Reported by The Lunala#6877

####Steps to reproduce:
- Disable Discord's permission to use the Nearby service from the Google settings app. 
- Open Discord. 
- Start nearby scanning and ask a friend near you to start the scanning too. 
- When you get the system modal that asks you if to give Discord permission to use Nearby service, hit allow. 
- Let the scanning continue for a while.

####Expected result:
The other friend using nearby gets detected eventually.
####Actual result:
The other friend never gets detected until you stop the search, close the add friends page and reopen page and restart search.
####Client settings:
Discord Alpha 856
####System settings:
Lenovo Vibe K5+, 5.1.1
