Latest Update: 2019-06-10T20:42:19.368Z

Reported by Alex1304#9704

####Steps to reproduce:
- Find a user that you have cached but that isn't in the server 
- Go to Search  
- Type `from:` 
- Input the user's tag

####Expected result:
The `from:` filter should work for any user you have cached no matter if they are in the server or not
####Actual result:
The `from:` filters only works for server members and as a result you can't search for messages from users that have left the server. This is possible on iOS.
####Client settings:
Discord Alpha 8.8.3
####System settings:
Huawei Mate 9 Android 9.0
