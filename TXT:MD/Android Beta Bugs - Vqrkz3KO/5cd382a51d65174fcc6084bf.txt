### Thanks for the report and image. This is not a bug and is intentional to prevent overlap of data. We had to choose between limiting the width of  embed field data/descriptions or pushing them down. Given the horiz limits of a phone I felt like it was better to allow desc/fields to have more horizontal space.

```
|  Title              | Thumb |
|                     | image |
-------------------------------
| Description                 |
| Embed data/fields           |
