####Steps to reproduce:
- Enter text that ends with a colon  
- Begin a code block without any preceding space  
- Enter text that contains a colon within the code block  
- i.e, "This is some sample text:```sample: with a colon```"

####Expected result:
The code block would display as a code block, with the "This is some sample text:" above it (as it does on Desktop).
####Actual result:
The code block is not displayed, instead showing the literal input with the backticks visible.
####Client settings:
9.0.9 (909)
####System settings:
Galaxy S6 Edge+, 7.0
