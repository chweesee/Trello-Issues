####Steps to reproduce:
- Open a server text chat 
- Hold down the members icon 
- Open the server draw while still holding the members icon 
- Tap on the different text chats 
- If there is a user profile or link behind the drawer it will act like the drawer does not exist and clicks on the profile or link.

####Expected result:
You should be able to click on the different text chats
####Actual result:
You click through the drawer and if there is a user profile or link it will click on that instead of going to the selected chat.

Beta 9.2.2
####System settings:
Google Pixel 3 Android version 9.0.0
