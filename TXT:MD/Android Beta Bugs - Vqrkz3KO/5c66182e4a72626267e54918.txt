Reported by Alex1304#9704

####Steps to reproduce:
- Go to Server Settings > Emoji 
- Click any emoji 
- Edit the name but DON'T hit Save 
- Tap the 3 dots 
- Delete emoji 
- When the unsaved changes warning appears, click Cancel

####Expected result:
It should either prevent the emoji from being deleted, or it should exit you from the editing screen
####Actual result:
You will stay in the editing screen but the emoji gets actually deleted, so if you attempt to save the changes or delete the emoji again, it will say "Unknown emoji"
####Client settings:
Discord Alpha 8.4.5G
####System settings:
Huawei Mate 9 Android 8.0
