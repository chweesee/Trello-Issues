####Steps to reproduce:
- Open Gallery and share something to Discord. 
- In the incoming share screen, select any channel name (note: Select a channel from the list/options available; do not search for any channel). 
- Click on the 'x' button. 
- Now, click on the text field and type the QS server filter (asterisk i.e. * ) in the text field and search for a server which has no spaces in it's name. 
- Click on any server name. 
- Observe that it is not centred. 
- Repeat steps 2, 3, 4 & 5 and compare results (but this time you should search for the channel name in the text field).

####Expected result:
In both trials, the server name is centred.
####Actual result:
The server name isn't centred in the first try but fixes in the second.

Discord Alpha 848
