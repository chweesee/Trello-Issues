Reported by Alex1304#9704

####Steps to reproduce:
- Go to a server 
- Go to Search 
- In the searchbar, type anything that is formatted as a Discord tag, eg `user#0000` 
- Notice that when you start typing the name, the search filters disappear, which is normal. Observe how they reappear as soon as you finish to type the 4th digit of the discriminator

####Expected result:
The search filters should remain hidden
####Actual result:
The search filters unexpectedly pop up again. This also happens when the search query is a random string starting with a # (which would correspond to a channel tag)
####Client settings:
Discord Alpha 7.5.2
####System settings:
Huawei Mate 9 Android 8.0
