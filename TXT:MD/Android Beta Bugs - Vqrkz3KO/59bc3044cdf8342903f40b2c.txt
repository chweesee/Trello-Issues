####Steps to Reproduce:
- Create a server or go to one where you are admin/owner 
- Create a new role with the teal/turquoise color(Go to server settings; then to roles section; create a role and give it the teal/turquoise color; save) 
- Close the app (or go to another server then come back; or just close the color picker) 
- Open the app/go back to server/reopen the color picker 
- Go to server settings, then to roles 
- Go to the role that you made previously, and open the color picker 
- Select another color (without closing the color picker); Observe the  check mark for the color that the role had previously is stuck until you reopen the color picker 
- (see attachment for info)

####Expected result:
The check mark should appear only for the current color of the role
####Actual result:
The check mark appear for the both colors; The old and current/new one until you reopen the color picker
####Client settings:
Discord Alpha 5.1.6
####System settings:
Android 4.4.4, Allview VivaH8LTE
