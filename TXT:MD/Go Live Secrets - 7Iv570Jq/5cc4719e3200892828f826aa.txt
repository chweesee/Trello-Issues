**Short Description: **Stream ended window doesn't disappear when clicking on somoene streaming
**Steps to reproduce:**
1. Watch someone streaming
2. Have them end their stream
3. Have them start streaming again in the same vc
4. Start watching their stream
5. Observe how it still says stream ended when you are watching a live stream
**Expected Result:** The stream ended window should close and the new stream should start
**Actual Result:** The stream ended window stays and the new stream does not start
**Client Settings:** Canary 36525
**System Settings:** Windows 10 Home x64 (1809)
