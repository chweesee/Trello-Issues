When you switch to a different stream while the popout is already open, the title of the popout containing the streaming user's name doesn't update. (https://trello.com/c/2SxFJlv8)

Investigating

Created: 2019-04-27T21:15:58.612Z
Latest Update: 2019-06-17T22:08:20.323Z

**Steps to Reproduce:**
1. Have someone go live in a server.
2. Watch the stream in popout mode.
3. Join someone else's stream.
4. Check the title of the popout window.

**Expected Result:** The popout title updates with the name of the user you are watching.

**Actual Result:** The popout continues to return the old user's username.

**Client Settings:** Canary 36529

**System Settings:** MacBook Pro (13", Mid-2012), macOS v10.14.4 (18E226)
